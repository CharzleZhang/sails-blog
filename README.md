## 开发者
**关于 [NodeJs](http://nodejs.org) **
```sh
* 安装依赖
$ npm install
```


**运行:**
```sh
* 挂起服务
$ sails lift
```

**生成文档:**
```sh
* http://127.0.0.1:1337/doc
$ npm install apidoc -g
$ npm run api
```

**创建sails:**
```sh

$ sails generate api <foo>

$ sails generate model <foo> [attribute1:type1, attribute2:type2 ... ]

$ sails generate controller <foo> [action1, action2, ...]
```
